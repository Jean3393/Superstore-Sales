# Superstore-Sales

Dashboard simples de vendas de uma loja fictícia dos EUA.

Análise contendo Indicadores de Vendas, receita por cidades, por estados e follow-up de vendas através dos anos.

## Fonte
Arquivo .csv com os dados retirados da plataforma Kaggle:
https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting/data
